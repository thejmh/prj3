# 테트리스 (Tetris)

바닐라 JS + HTML5 Canvas로 만든 가벼운 테트리스. 빌드 과정·의존성 없음. `index.html` 파일 하나가 전부입니다.

## 실행 (로컬)

그냥 `index.html`을 브라우저로 열면 됩니다. 또는 간단한 정적 서버:

```bash
npx serve .
```

## 조작

| 키 | 동작 |
|----|------|
| ← → | 좌우 이동 |
| ↑ (또는 X) | 회전 |
| Z | 반대 회전 |
| ↓ | 소프트 드롭 |
| Space | 하드 드롭 |
| C | 홀드 |
| P | 일시정지 |

모바일: 좌우 스와이프 이동, 아래 스와이프 소프트드롭, 탭 회전, 길게 아래로 스와이프 하드드롭.

## 기능

- 7-bag 랜덤 방식
- 고스트 피스, 홀드, 다음 조각 미리보기
- 라인 클리어 점수(1/2/3/4줄 = 100/300/500/800 × 레벨)
- 레벨업에 따른 낙하 속도 증가

## Vercel 배포

정적 사이트라 설정이 필요 없습니다.

```bash
vercel        # 프리뷰 배포
vercel --prod # 프로덕션 배포
```

또는 GitHub 저장소에 push 후 Vercel 대시보드에서 import → 프레임워크 프리셋 "Other" 그대로 두면 됩니다.
